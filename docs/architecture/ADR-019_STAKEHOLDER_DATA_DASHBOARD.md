# ADR-019: Gold Tier Impact Intelligence Dashboard

## Status: Proposed (Tier 3)

## Context
To secure long-term sustainability, HALA needs quantifiable proof of impact. Manual tracking of "The Peoples Guide" usage is impossible.

## Decision
We will architect a **Server-Side Aggregation Layer** that:
1. **Aggregates:** Anonymized events (Searches, Calculator results, Call-clicks).
2. **Visualizes:** A private, authenticated Web Portal -- Impact Dashboard, Hunger Heatmap.
3. **Exports:** Schema-ready CSV/PDF assets for Grant compliance.

## Logic
This ensures HALA can say: "Our $150k Gold investment resulted in $X Million of federal aid access for Angelenos."
